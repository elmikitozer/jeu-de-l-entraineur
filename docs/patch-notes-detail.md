# Notes de mise à jour (détaillé) — Jeu de l'Entraîneur

---

## 12 juin 2026

### 🆕 Page détaillée par match

Cliquer sur n'importe quel match (depuis l'accueil ou l'onglet **Matchs**) ouvre une page dédiée avec :

- Le score en grand, le statut (en cours / terminé / à venir), la phase et le stade
- Les **temps forts** : buts, cartons et passes décisives, avec le nom du joueur et l'équipe
- Les **compositions** : joueurs de chaque équipe ayant joué, avec leurs points fantasy, cliquables pour accéder à leur fiche
- Les **points fantasy générés** sur ce match : détail par joueur (but, victoire, MOTM…) et les participants qui l'ont sélectionné
- L'**impact sur le classement** : gain ou perte de points par participant
- Navigation **Précédent / Suivant** entre les matchs

### 🐛 Correction — Page joueur

La page de chaque joueur affichait 0 point. Elle affiche maintenant le total correct et l'historique match par match avec le détail des points.
